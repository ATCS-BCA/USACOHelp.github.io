What is USACO?

Which Language?

The most popular language for USACO work is likely C++. This tutorial begins with Java as that is the language most beginers know best. Once you get a hang of USACO, you are more than welcome to learn C++, even encouraged. However, never fear if you decide to stick with Java. All of the USACO problems can be successfully written in either language.

Environment Setup

Here is a video tutorial for getting setup.

  1. Clone the repo https://github.com/ATCS-BCA/USACOHelp.github.io.
  2. Open USACO_Intro in your favorite IDE. Our demo uses VSCode. More VSCode instructions are listed below.
  3. Run section1/section1_1/test.java. It should produce a new file test.out. If it does, your environment is setup!
